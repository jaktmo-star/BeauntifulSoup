# 🕸️ Raspagem de Dados com Python

Projeto simples de **Web Scraping** utilizando **Python** e **BeautifulSoup** para capturar informações de produtos em uma página HTML.

---

## 📚 Tecnologias Utilizadas

* Python 3
* BeautifulSoup4

---

## 📦 Instalação

Instale a biblioteca necessária:

```bash
pip install beautifulsoup4
```

---

## 📁 Estrutura do Projeto

```bash
📂 projeto
 ├── index.html
 ├── BeautifulSoup.py
 └── README.md
```

---

## 🚀 Como Executar

1. Coloque o arquivo `index.html` na mesma pasta do script Python.
2. Execute o arquivo:

```bash
python BeautifulSoup.py
```

---

## 🔍 Funcionalidades

✔ Leitura de arquivo HTML
✔ Busca automática de produtos
✔ Captura do nome do produto
✔ Captura do preço do produto
✔ Exibição organizada no terminal

---

## 💻 Exemplo de Saída

```bash
------------------------------
O nome do produto é: Notebook Gamer
O preço do produto é: R$ 4.500

------------------------------
O nome do produto é: Mouse RGB
O preço do produto é: R$ 120
```

---

## 🧠 Conceitos Utilizados

* Web Scraping
* Manipulação de HTML
* Estrutura de repetição (`for`)
* Uso de funções `find()` e `find_all()`
* Tratamento de texto com `.strip()`
